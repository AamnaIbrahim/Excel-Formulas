# Excel Formulas Practice Repository

This repository contains practical examples of commonly used Microsoft Excel formulas. Each formula is demonstrated using a sample employee dataset to help beginners understand how the functions work in real-world scenarios.

## Included Formulas

### Math Functions
- SUM
- SUMIF
- SUMIFS
- MIN
- MAX

### Text Functions
- CONCATENATE
- LEN
- LEFT
- MID
- RIGHT
- TRIM
- SUBSTITUTE
- Date to Text

### Logical Functions
- IF
- IFS

### Counting Functions
- COUNT
- COUNTIF
- COUNTIFS

### Date Functions
- DAYS
- NETWORKDAYS

### Lookup Functions
- VLOOKUP
- XLOOKUP
- XLOOKUP (Horizontal)


## Dataset

The examples use a sample employee dataset containing:

- Employee ID
- First Name
- Last Name
- Age
- Gender
- Job Title
- Salary
- Start Date
- End Date

This dataset is used to demonstrate how different Excel formulas work with real data.
